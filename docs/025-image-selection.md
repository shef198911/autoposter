# Image selection rules

Social platforms often need a public image URL.

## Preferred order

For articles, prefer social images that are already prepared for sharing:

```txt
socialImage → social_image → ogImage → og_image → cover
```

For shorts, prefer the short-specific image first:

```txt
image → socialImage → cover
```

## Recommended format

Use JPG or PNG for platform publishing.

Instagram is stricter than Telegram and Bluesky, so a social JPG is usually the safest option.

## Public URL requirement

The image URL must be reachable by the platform API without login or cookies.
