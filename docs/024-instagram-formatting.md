# Instagram formatting rules

Instagram captions should be readable even when the link is not clickable inside the caption.

## Recommended caption shape

```txt
Title

Summary

Read more: https://example.com/post-slug

#CoinBlog #Crypto #Web3 #Blockchain
```

## Rules

- Use a public image URL.
- Keep the caption clear and not too long.
- Include the article URL as plain text.
- Add a small hashtag set.
- Avoid unsupported image formats such as SVG.

## Test first

Always use preview and dry-run before a real Instagram publish.
