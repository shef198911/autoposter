# Post-publish checklist

Use this checklist after sending real posts.

## Review results

- Confirm each target platform received the post.
- Check the published title, summary, link, and image.
- Review the command output for errors.
- Save any useful logs for debugging.

## If something failed

Retry only the affected platform or slug when possible.

Avoid running a full bulk send again until you understand which items were already posted.

## Duplicate safety

Do not delete local posting history unless you intentionally want to allow reposts.
