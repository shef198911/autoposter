# Message formatting rules

Message formatting should stay predictable across platforms.

## General rules

- Keep titles short and readable.
- Preserve paragraph breaks where the platform supports them.
- Use platform-specific links and footers.
- Avoid exposing internal feed fields in public posts.

## Telegram

Telegram messages can use HTML links and escaped text.

## Bluesky

Bluesky posts should stay short and use external links where appropriate.

## Instagram

Instagram captions can be longer, but should remain readable and include a public article link.
