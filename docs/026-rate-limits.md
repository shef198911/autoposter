# Platform rate limits

Social platforms can limit how often an app publishes content.

## Practical rules

- Avoid publishing a large backlog without testing.
- Start with one post per platform.
- Use dry-run before real sends.
- Keep logs for failed API responses.

## Instagram

Instagram Graph API has publishing limits and stricter media checks.

Use one real test post before running bulk sends.

## Telegram

Telegram is usually tolerant, but rapid repeated sends can still look spammy.

## Bluesky

Keep text short and avoid retry loops after failed sends.
